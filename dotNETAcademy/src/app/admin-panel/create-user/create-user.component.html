Create User

  1. Maak een nieuwe user in Azure B2C aan.
  2. Kopieer het Object ID van de user vanuit Azure.
  3. Plak het Object ID hieronder.
  4. Kies of het een user of admin moet worden.
  5. Klik op aanmaken.
